# PixivSpider
## A Spider For Searching Hot Pictures From Pixiv.net

###Usage
- Tools:      Python3 & Scrapy
- Usage:      Enter your e-mail, password and params in PixivSpider/settings.py, then run main.py
- Example:
Params:       's_mode=s_tag&word=perfume'
